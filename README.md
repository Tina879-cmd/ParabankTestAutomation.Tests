# Parabank Test Automation

## Setup

1. Install .NET 10 SDK
2. Run `setup.ps1` to install dependencies
3. Update `appsettings.json` as needed

## Execution

- Run tests: `dotnet test`
- Generate Allure report: `allure generate --clean`

## Features

- BDD with SpecFlow
- Playwright browser automation
- Allure reporting
- FluentAssertions
- Azure DevOps CI/CD
