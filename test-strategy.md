# Test Strategy

## Scope

- Functional testing of Parabank web app
- Authentication, validation, negative testing

## Approach

- BDD with Gherkin (Reqnroll)
- Page Object Model
- Synchronization and robust error handling
- Logging and Playwright HTML reporting

## Tools

- .NET 10, Playwright, Reqnroll, FluentAssertions

## CI/CD

- GitHub Actions pipeline
